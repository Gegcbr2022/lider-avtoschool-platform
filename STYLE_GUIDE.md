# Style Guide

Проєкт: автошкола «Лідер»
Дата: 2026-06-02

## Colors

| Token | Value | Use |
| --- | --- | --- |
| Primary red | `#ff1e1e` | Main CTA, active states, icons, important accents |
| White | `#ffffff` | Page background, cards, form surfaces |
| Light gray | `#f4f4f4`, `#f5f5f5` | Section background, soft panels, secondary surfaces |
| Text | `#1a1a1a` | Primary headings and body text |
| Muted | `#666666` | Descriptions, helper text, secondary labels |
| Line | `#e5e5e5` | Borders and dividers |

## Typography

- Headings: bold, compact, commercial tone; no negative letter spacing beyond existing framework defaults.
- Body text: 16-18px on marketing sections, clear line-height, short paragraphs.
- Mobile: avoid hero copy that pushes all useful actions below the first screen.

## Buttons

- Primary CTA: red background, white text, rounded 16-24px, minimum tap target 44px.
- Secondary CTA: white or dark graphite background with strong contrast.
- CTA labels must be action-led: `Записатися`, `Подзвонити`, `Telegram`, `WhatsApp`, `Отримати консультацію`.

## Cards

- Radius: 18-24px for commercial cards, 8px minimum where dense UI is needed.
- Avoid nested cards unless the inner element is a real repeated item.
- Use subtle borders and soft shadow; do not overuse heavy floating sections.

## Blocks

- First viewport must show the offer, real/generated relevant image, price hint, call and signup action.
- Every major section should support one of the ad funnel goals: trust, price clarity, branch choice, quick contact, lead form.
- Technical terms are not shown in client UI. Use human wording: learning, documents, practice, manager, branch, consultation.

## Mobile

- Mobile is primary. CTA must be reachable with one hand.
- Keep popup close button sticky and visible.
- Forms use large fields, simple labels, and one clear submit button.
- Floating widgets must not overlap the bottom quick-action bar.

## Social Colors

- Facebook: `#1877F2`
- Instagram: gradient `#f58529` -> `#dd2a7b` -> `#515bd4`
- YouTube: `#FF0000`
- Telegram: `#229ED9`
- WhatsApp: `#25D366`
