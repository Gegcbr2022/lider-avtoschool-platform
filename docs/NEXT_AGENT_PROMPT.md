# Наступний крок для Opus 4.8 / Codex

Ти — архітектор та старший розробник.

Gemini успішно оновив UI/UX для:
- Головного екрана (перетворено на дашборд з великими картками).
- Навчання (перетворено на покрокову дорожню карту з Лідіком).
- Рейтингу ПДР (empty states стали більш мотивуючими).
- Профілю (полагоджено "вічний бейдж 1").

## Завдання на наступний спринт:
1. Забезпечити працездатність "Тренажера ПДР" (модуль `pdr-questions.ts` та екран `tests.tsx`). Інтерфейс підготовлений, але потрібно довести до ідеалу "Game Loop" (дуелі ПДР, слабкі теми, агрегація результатів).
2. Налаштувати бекенд-логіку для агрегації "Рейтингу ПДР" за тиждень/місяць (наразі UI тимчасово приховує фільтри часу).
3. Зробити фінальну збірку APK, прогнати `adb monkey` і закрити **S2 / S3 спринти**.

Починай з рев'ю `tests.tsx` та зв'язку з Firebase!
