# Mobile UX Audit — Автошкола «Лідер» (2026-06-10)

Цель: уровень Monobank/Duolingo/Apple-like liquid UI, бренд-акцент `#ff1e1e` (в коде
`red=#E51D1D` dark / `#DF1616` light). Без инфантильности, без «заглушек».

## 1. Главный экран — `app/(tabs)/index.tsx`

**Было:** quick-actions = 4 квадрата `flex:1`, эмодзи 24px + подпись 11px, ~76px ширина.
Ощущение «сжато/ущербно/без вау» (жалоба №11).

**Сделано (этой сессией):** 2×2 сетка премиум-карточек:
- иконка в скруглённом тайле 46×46 с мягкой заливкой (`tintSoft`),
- цветной шеврон-акцент,
- заголовок 15 (900), подзаголовок 12 (Тренажёр персонализирован: `Продовжити · {bestScorePct}%` / `500 питань · іспит МВС`),
- `minHeight:116`, корректные отступы, тема-безопасно (только токены), ок на 360px.

**Дальше (Sonnet, опц.):**
- Тренажёр сделать «героем» (full-width карточка с кольцом прогресса + «слабкі теми»).
- Микро-анимация появления (stagger fade/scale), haptics на нажатии.
- «До екзамену N днів» бейдж на карточке учня, если задана дата.

## 2. Рейтинг ПДР — `app/(tabs)/club.tsx:942` `LeaderboardView`

**Факты:** данные реальные (`getLeaderboard(30)`, без мока). Сортировка chips: Точність/Відповідей/Серія.
Топ-3 — увеличенный медальный emoji в строке; «ВИ» подсветкой. Empty: «Рейтинг порожній…».

**Проблемы (подтверждено на живом экране):**
1. Solo-state: один участник = ты, 🥇 #1 с 30% — выглядит пусто и обесценивает медаль.
2. Нет подиума топ-3 (только строки списка).
3. Текущий юзер НЕ закреплён — если ты #25, нужно скроллить, чтобы себя найти.
4. Только критерий сортировки, нет окон **тиждень / місяць / весь час**.
5. Несправедливость: 100% с 5 ответов > 95% с 500 ответов (сортировка по accuracy без порога).
6. Акцент янтарный (`colors.warning`), а не бренд-красный.
7. Нет мотивации «догнати наступного» / месседжа от Лідіка.

**Целевой дизайн (для Sonnet/Codex):**
- **Подиум** топ-3 (центр выше, 🥇🥈🥉, аватар-кружок, % крупно), бренд-красный/золото аккуратно.
- **Список 4..N** компактными строками.
- **Закреплённая нижняя плашка «Ваше місце: #N»** (sticky), с дельтой до следующего: «+12 відповідей щоб обігнати …».
- Переключатель **окон** (Тиждень/Місяць/Всі) + критерий (Точність/Відповіді/Серія).
- **Честность:** для accuracy — порог `totalAnswered ≥ 20`, иначе в отдельную лигу «Новачки».
- **Solo/empty:** «Ти поки єдиний у рейтингу — запроси друзів і позмагайтесь» + кнопка Share (есть `handleReferral`).
- Реплика Лідіка: «До топ-10 лишилось N правильних відповідей. Поїхали!».
- Пустые данные: дружелюбный empty-state, НЕ выдавать одинокого юзера как чемпиона.

## 3. Профиль — `app/(tabs)/profile.tsx`

- **Бейдж «1»** — исправлен в `_layout.tsx` (см. BUGS B-01). Профиль концептуально не место для
  «непрочитанных нуджів».
- Хорошо: hero с аватаром/эмодзи, stats-полоса, sheets (Особиста/Налаштування/Сповіщення/Безпека/Підтримка),
  email-verify баннер, dev-режим по 10 тапам версии.
- Улучшить (опц.): «Прогрес профілю N%» полоса (заповни телефон/місто/категорію/документи) — но БЕЗ
  навязчивого бейджа; «Документи для НАІС» статус-чип.

## 4. Темы — `lib/themeContext.tsx`

Палитры light/dark состоятельны (OLED-чёрный / off-white, контраст текста ОК). Системного бага
контраста нет (был починен в коммите `572ee7f`). Действие: Sonnet — spot-check `tests.tsx` (45 хардкодов
цвета — большинство легитимны на цветном фоне ответов, но проверить нелегитимные), и пройтись по всем
экранам в обеих темах через ADB-тоггл (Профіль→Налаштування→Тема).

## 5. Навигация/таб-бар — `app/(tabs)/_layout.tsx`

5 вкладок: Головна/Навчання/Чат/Клуб/Профіль. Скрытые роуты (tests/practice/assistant) через push — ок.
Бейдж чата корректно считает только чужие сообщения. Touch-таргеты таб-бара ОК (height 72 android).

## Чек-лист приёмки UX (для финального review)

- [ ] Главная: 2×2 карточки крупные, читабельные, ок light/dark, ок 360px.
- [ ] Бейдж профиля не горит без реальных actionable-уведомлений.
- [ ] Рейтинг: подиум + закреплённая позиция + окна + честность + solo-CTA.
- [ ] Все экраны проверены в light и dark.
- [ ] Украинский текст без обрезаний/переполнений.
