# LidikGuide Component

Цей компонент створено для впровадження Лідіка (маскота-помічника) як провідника по всьому застосунку.

## Можливості
- Підтримка варіантів `default` (бульбашка збоку), `card` (в карточці з тінями) та `inline` (горизонтальна плашка).
- Вбудована підтримка тем (використовує кольори `colors.infoSoft` та `colors.bgElevated`).
- Можливість додати Action Button (CTA).
- Може працювати з емодзі `🤖` як placeholder, доки не буде надано повноцінну графіку.

## Використання
```tsx
import { LidikGuide } from "../../components/lidik-guide";

<LidikGuide 
  variant="inline"
  text="Почни з маленького кроку — 5 хвилин тренування вже дадуть результат."
/>
```
