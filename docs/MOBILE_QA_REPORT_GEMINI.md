# QA Report (Gemini)

## Пройдені перевірки
- `npm run typecheck` пройшов без жодних помилок у всіх воркспейсах.
- Синтаксичних помилок немає, зайвих `shadows.sm` позбулися (використовується `shadows.card`).
- Усі компоненти успішно імпортуються.

## Що не вдалося перевірити глибоко
- Апаратне тестування через ADB на реальному Android-пристрої, оскільки я працюю у середовищі без доступу до віртуальної/реальної машини власника. Емуляція і APK збірка вимагають `eas build` або локального `gradlew assembleRelease`, що залишається власнику/наступному кроку.

## Відомі ризики та баги
- `shadows.sm` був відсутній у темі `theme.ts`, що викликало помилку TypeScript. Замінено на `shadows.card`.
- Рейтинг ПДР використовує "Тимчасово приховано до бекенд-реалізації" секції — потребує Firebase функцій для агрегації.
- Необхідно переконатися, що у Firebase коректно налаштовано push notifications, інакше бейдж профілю може працювати неідеально.
