# Agent Tasks — Claude Sonnet

Sonnet = UI/UX и контент. Аккуратные патчи, низкий риск. Работает ПОСЛЕ Codex по тем же файлам,
не конфликтует с логикой. Цель — Monobank/Duolingo/Apple-like ощущение, бренд `#ff1e1e`, без инфантильности.

## Жёсткие правила
- Только токены темы (`useTheme().colors`, `radii`, `spacing`, `shadows`) — НЕ хардкодить цвета,
  если это не текст на заведомо цветном фоне.
- Проверять КАЖДЫЙ экран в light и dark (ADB-тоггл: Профіль→Налаштування→Тема).
- Украинский основной, тексты без обрезаний; touch-таргеты ≥44px.
- Не менять бизнес-логику/Firestore; только разметка/стили/копия/анимации/empty-states.
- После: `npm run typecheck --workspace @lider/mobile`; ADB-скриншоты до/после.

## Очередь
1. **Главная — полировка** `index.tsx`: Тренажёр сделать «героем» (кольцо прогресса, «слабкі теми»);
   stagger-анимация появления карточек; haptics; бейдж «до екзамену N днів» на карточке учня.
2. **Рейтинг — визуальный пасс** (после рерайта Codex): типографика подиума, микроанимации, чистый solo-state.
3. **Empty-states везде**: чат, клуб-стрічка, нагороди, бронь — дружелюбно, с CTA, без «заглушек».
4. **Тексты UA** + микрокопия (кнопки, ошибки, подсказки Лідіка) — теплее, конкретнее.
5. **Темы spot-check** `tests.tsx` (45 хардкодов — найти нелегитимные на нейтральном фоне).
6. **Web** (опц.): hero/CTA/доверие/SEO-микрокопия в `apps/web` — только контент/стиль.
