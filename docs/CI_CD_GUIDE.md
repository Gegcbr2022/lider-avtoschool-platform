# CI/CD Guide

CI/CD — это автоматическая проверка проекта на GitHub.

## Что уже настроено

Файл:

```text
.github/workflows/ci.yml
```

Он запускает:

```bash
npm ci
npm run lint
npm run typecheck
npm run test
npm run build
```

## Как проверить на GitHub

1. Откройте репозиторий на GitHub.
2. Нажмите вкладку `Actions`.
3. Откройте последний запуск `CI`.
4. Убедитесь, что все шаги зелёные.

## Возможные ошибки

Если ошибка на `npm ci`:

- проверьте, что `package-lock.json` загружен в GitHub.

Если ошибка на `build`:

- откройте лог ошибки;
- найдите красную строку;
- исправьте файл, который указан в ошибке.
